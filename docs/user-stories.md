# User Stories

## Onboarding
- As a new user I can sign up with email + password, or continue with Google, and land on an empty dashboard.
- On signup the app pre-seeds a set of common income and expense categories so I can add my first transaction immediately.

## Accounts
- I can add a cash, bank, wallet, card, or credit-card account with a name, currency, and opening balance.
- I see the current balance of each account update instantly after every transaction.
- I can delete an account and its transactions in one action.

## Categories
- I have sensible default categories on day one and can add my own.
- I can browse expense and income categories separately.

## Transactions
- I can quickly record an expense, an income, or a transfer between two of my accounts.
- I can search and filter my transactions by kind and account.
- I can delete a transaction and my balances reconcile automatically.

## Dashboard
- I always see net worth across all accounts at the top.
- I see how much I earned and spent this month, and whether I ended up net-positive.
- I see a cash-flow chart for the last six months.
- I see which categories I spent the most on this month.

## Settings
- I can change my display name and base currency.
- I can sign out from any device.
