# Roadmap

## v1 (this release)
- Auth, accounts, categories, transactions, transfers, dashboard, settings.

## v1.1 — Refinement
- Edit transaction sheet.
- Per-account detail page with running balance + sparkline.
- CSV import & export.
- Better mobile quick-add (drawer instead of dialog).

## v2 — Planning
- Budgets per category with monthly rollover.
- Savings goals with progress bars.
- Recurring transactions engine + bill reminders.
- Subscription tracker.

## v2.5 — People & credit
- Debts / loans (I owe / owed to me) with settlement flow.
- Debtors / creditors directory.

## v3 — Intelligence
- AI-powered categorization (Lovable AI).
- Duplicate detection, suspicious spending alerts.
- Natural-language transaction entry ("I spent Rs 500 on lunch").
- Receipt OCR via Lovable AI vision models.
- Monthly forecast / prediction.

## v4 — Multi-workspace & business
- Personal / Business / Family workspaces with roles.
- Invoicing, tax categories, P&L / balance sheet reports.
- Family sharing with read-only members.

## v5 — Native
- iOS + Android via Capacitor from the same PWA codebase.
- Biometric lock, background sync.
